# 充能台

![充能台](item:actuallyadditions:block_empowerer)
![物品展示台](item:actuallyadditions:block\_display\_stand)
充能台是一種多方塊結構機器，用於增強某些物品，通常是水晶。

將充能台放下，然後在每個方向放置一個物品展示台，在它和充能台之間有 2 個方塊。
給物品展示台供電，按照配方放入物品，它就會開始製作。
