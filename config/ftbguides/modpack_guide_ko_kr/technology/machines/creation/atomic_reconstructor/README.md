# 원자 재조합기

![원자 재조합기](item:actuallyadditions:block\_atomic\_reconstructor)
원자 재조합기(Atomic Reconstructor)는 RF를 사용해 아이템을 다른 아이템으로 변환합니다.
주로 다양한 기계의 재료로 사용되는 여러 결정들을 만들 때 사용합니다.
또한 모래를 영혼 모래로 만들거나 네더 석영을 프리즈마린으로 변환하는 등 여러 다른 사용법들도 있습니다. 원자 재조합기에 커서를 올리고 U키를 눌러 모든 가능한 조합법들을 확인하세요.

원자 재조합기는 레이저를 발사해 맞은 아이템을 변환하는 방식으로 작동합니다.
기본적으로는, 에너지가 남아있다면 주기적으로 레이저를 발사합니다. 레드스톤 횃불을 들고 기계를 우클릭해 레드스톤 신호가 들어올 때만 작동하도록 모드를 바꿀 수 있습니다.
![이런식으로 발판을 붙여서 설치하면 발판에 아이템이 떨어질 때만 작동해 아이템을 변환하므로 자동화하기에 유용합니다.](atomic_reconstructor.png)

![렌즈](item:actuallyadditions:item_misc 1 18)
원자 재조합기에 렌즈를 장착해 기능을 변경할 수 있습니다. 렌즈들은 레이저에 폭팔하게 만들거나, 몬스터를 죽이거나, 블록의 색을 바꾸거나, 아이템에 적용된 인챈트를 책으로 옮기거나, 광물은 생성하는 등의 기능을 부여합니다.
