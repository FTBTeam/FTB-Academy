# 몬스터 처리

이제 몬스터를 자동으로 생성할 수 있으니, 몬스터 처리 과정을 자동화할 차례입니다.

![Spikes](item:mob\_grinding\_utils:spikes)
Spikes는 어떤 몬스터든 접촉하면 천천히 피해를 줘서 죽입니다. 몬스터는 경험치를 드롭하지만 블레이즈 막대처럼 플레이어가 죽여야만 나오는 아이템은 드롭되지 않습니다.

![Mob Masher](item:mob\_grinding\_utils:saw)
Mob Masher는 레드스톤 신호가 들어왔을 때 접촉한 몬스터에게 피해를 줍니다. Mob Masher에는 전용 업그레이드 슬롯이 있습니다. 완전히 업그레이드되면 날카로움 5, 약탈 5, 참수 5 효과를 가질 수 있습니다.

![몹 분쇄기](item:draconicevolution:grinder)
드라코닉 에볼루션(Draconic Evolution) 모드의 (Mob Grinder)는 RF를 사용해 정면 범위 내의 몹을 즉시 죽입니다. 위더도요. 플레이어가 죽여야만 나오는 아이템들도 정상적으로 드롭됩니다.

![몹 파쇄기](item:industrialforegoing:mob_relocator)
몹 파쇄기(Mob Crusher)는 작동 범위 내의 몬스터를 죽이고 플레이어가 죽여야만 나오는 아이템을 제공합니다. 몬스터에게서 드롭된 아이템은 내부 저장공간으로 이동되며, 경험치는 몹 복제기(Mob Duplicator)에서 사용하는 숨결(Essence)로 변환되어 저장됩니다.
기본 작동 범위가 1x1이기 때문에, 범위 애드온을 적용시켜 작동 범위를 늘릴 필요가 있습니다.

![몹 분해기](item:industrialforegoing:mob\_slaughter\_factory)
몹 분해기(Mob Slaughter Factory)는 작동 범위내의 몬스터를 죽입니다. 하지만 몬스터가 죽으면 아이템과 경험치를 드롭하는 대신 액체 고기가 생산된다는 차이가 있습니다. 또한 분홍 슬라임 액체를 조금 생성하기도 합니다.
