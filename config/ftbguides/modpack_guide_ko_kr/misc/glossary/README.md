# 중요한 용어들

모드팩(modpack)이란 미리 만들어진 모드 묶음입니다. 모드팩마다 다른 모드 구성을 가지거나 특별한 무언가를 가지고 있습니다.

kitchen sink modpack이라고 불리는 모드팩들은 게임 플레이상의 변화 없이 모드들을 묶어놓기만 한 모드팩을 가리킵니다. 이름의 유래는 "everything but the kitchen sink." 라는 구절입니다(상상할 수 있는 모든 것, 거의 모든 것 등의 의미를 가짐).

expert modpack이라고 불리는 모드팩들은 더 어렵도록 조정된 모드팩을 가리킵니다. 보통 아이템 조합법들을 바꾸어 더 길고 어려운 과정을 거치도록 만듭니다.

RF(Redstone Flux)와 FE(Forge Energy)는 거의 대부분의 기술 모드들이 사용하는 범용 에너지 시스템의 이름입니다. 이 모드팩에 포함되어 있지는 않지만 엔더 IO(EnderIO)모드의 μI(microinfinity)와도 호환됩니다. RF 전송량은 보통 틱당 RF 전송량이라는 의미를 가지는 RF/t으로 표기됩니다.

청크(chunk) 가로 16, 세로 16, 월드 바닥부터 천장까지의 크기를 가진 세계를 나누는 단위입니다. F3+G를 눌러 청크 경계를 확인할 수 있습니다.

틱(Tick)이란 게임이 실행되는 속도입니다. 최적의 조건에서(랙이 없는 상태), 1틱은 1/20초입니다. 랙이 발생하면 이 한 틱의 주기가 늘어나 초당 틱 비율(tick rate)이 내려갈 수 있습니다. "/forge tps" 명령어를 입력해 현재 디멘션의 초당 틱 비율을 확인할 수 있습니다.
